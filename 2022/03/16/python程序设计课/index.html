python程序设计课

第三周才选上课，第四周去核酸检测了，还没上过课。

看一下老师的PPT。

输入

<变量>= input(<提示性文字*>)

可在input前强制类型转换

输出

print(<表达式1>[, … , <表达式n>, sep=' ', end='\n'])

默认间隔符为空格，结束符为换行

格式化输出：

print(格式字符串 % (表达式1,…,表达式n))

格式字符串.format(表达式1,…,表达式n)

随机数

print('%.3f'%1.23456)

会四舍五入

数据类型

python的整数没有长度限制

python整数支持四种计数制

  • 0x 开头表示16进制
  • 0b开头表示2进制
  • 0o开头表示8进制

python复数定义与数学一致，z = a + bj j为虚部表示

对于复数z,可以用 z.real获得实部，z.imag 获得虚部

不同的数据类型进行混合运算时，会发生隐式类型转换，（低向高）

一些内置函数

Math模块中的函数

函数 描述
fabs(x) 返回x的绝对值（float类型）
ceil(x) 向上取整，返回大于等于x的最小整数
floor(x) 向下取整，返回小于等于x的最大整数
trunc(x) 返回x的整数部分
factorial(x) 返回整数x的阶乘
sqrt(x) 返回x的平方根
exp(x) 返回e的x次幂，e是自然对数
log(x) 返回x的对数值，即ln x
常量 数学表示 描述
math.pi π 圆周率，值为3.141592653589793
math.e ℮ 自然对数，值为2.718281828459045
math.inf ∞ 正无穷大，负无穷大为-math.inf
math.nan 非浮点数标记，NaN（Not a Number）
库的导入

  1. ​ import math

    ​ 对math库中函数采用 **math.<函数名**>()的形式

  2. from math import

    ​ 直接调用函数
